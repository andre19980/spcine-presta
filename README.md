# SpcinePresta
Programa para executar análise e gerar relatórios de prestação de contas de proponentes da Spcine.

**Pré-requisitos:**

O programa foi desenvolvido em Python com o framework Streamlit. A aplicação encontra-se hospedada na própria plataforma em nuvem do Streamlit para a comunidade.

- Python >= 3.12.4
- Streamlit >= 1.52.1
- Authlib >= 1.3.2
- Openpyxl >= 3.1.2
- Fpdf2 >= 2.8.5
